## 游客模式实现提示（基于现有项目）

- 游客可用功能（全部本地处理，不触云）：扫描设备、BLE 控制设备、设置页面、浏览仪表盘。
- 游客限制：不支持云同步；未绑定邮箱前不支持跨设备登录。
- 统一提示语（按场景使用）：
  - 进入游客模式时：`当前为游客模式，仅支持本地设备控制，云同步不可用。`
  - 访问云同步/账号相关功能时：`该功能需登录正式账号（先绑定邮箱后可跨设备同步）。`
  - 需跨设备登录时：`游客模式无法跨设备登录，请绑定邮箱并登录正式账号。`

## 游客功能列表
| 功能        | 游客可用？ | 备注       |
| --------- | ----- | -------- |
| BLE 扫描与连接 | ✔     | 不依赖用户身份  |
| 设备控制      | ✔     | 与本地设备通信  |
| 首页仪表盘     | ✔     | 完全本地     |
| 修改密码      | ❌     | 游客无密码    |
| 第三方绑定     | ❌     | 必须先注册账户  |
| 清除缓存      | ✔     | 系统级逻辑    |
| 关于、协议     | ✔     | 静态内容     |
| 账号删除      | ❌     | 游客账号不能删除 |
