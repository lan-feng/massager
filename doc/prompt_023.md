# 设备组合问题修改
扫描项目，解决以下问题
# 1.扫描问题
扫描屏蔽问题，主页进入扫描页面只屏蔽主页列表的设备信息，不能屏蔽已经在combo_info的设备；设备控制进入扫描页面只屏蔽当前主设备combo_info的设备，不能屏蔽已经在主页列表的设备信息。
# 2.设备控制页面问题
1.进入设备控制页面，蓝牙连接需要实时切换选中设备的蓝牙。例如：选中设备A 就连接A设备的蓝牙，切换到设备B 就连接B 设备的蓝牙；对应的控制面板也要实时切换，确保接收和发送的信息都是选中设备的信息。
2.退出设备控制页面后，断开相应的蓝牙连接。
3.在设备控制页面，选中非主设备长按时弹出管理功能，提供删除附属设备和重命名附属设备；然后更新主设备的combo_info。
4.控制设备组管理控件的高度和每个设备的宽度，避免水平滚动时应为高度问题出现屏幕的拉伸和缩小。
