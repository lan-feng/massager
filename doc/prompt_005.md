Implement a Bluetooth Low Energy (BLE) manager in Kotlin for the MASSAGER app.

Features:
- Scan for nearby devices with names containing “MASSAGER” or “SmartPulse”.
- Display found devices in a list with signal strength.
- Connect to a device when selected.
- Maintain connection state (connected/disconnected).
- Provide callbacks for connection events.
- Use Android BluetoothAdapter and BluetoothLeScanner APIs.
- Prepare for later communication via GATT services (read/write characteristics).
