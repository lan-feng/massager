Home Screen
Build the "Home" screen for the MASSAGER app.

Features:
- Title: “Welcome to MASSAGER”.
- Top right corner: “+” icon to add a new device.
- Section title: “Common Device”.
- Show a device card (example: SmartPulse TENS Unit):
  - Device icon (circular image)
  - Device name: “SmartPulse TENS Unit”
  - Subtitle: “SmartPulse TENS Device”
- On clicking a device card, navigate to the device control page (not implemented yet).
- Bottom navigation bar with 4 tabs:
  1. Home (red icon)
  2. Manual / Guide
  3. Device list
  4. Profile / Me
- On-screen notification bubble (toast-like) saying “Success” after device added.
- Use soft colors, shadowed cards, rounded corners, and white background.
- Include pull-to-refresh for the device list.

Make the layout adaptive for phones and tablets.
